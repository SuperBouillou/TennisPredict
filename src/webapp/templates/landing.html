Le modèle

La calibration

Platt scaler recalibré sur les cotes Pinnacle no-vig (2023–2024). Edge = probabilité modèle − probabilité implicite Pinnacle. Si le modèle dit 65% et Pinnacle implique 55%, l'edge est +10%. On ne signale que les vraies divergences.

Edge ≥ 5%
[f(x)]
Kelly Criterion

Mise proportionnelle à l'edge et au bankroll. Quarter-Kelly avec cap à 5% par pari. Gestion du risque mathématique — pas émotionnel. Backtesté sur données réelles hors-entraînement.

// Comment ça marche
01
Scraping des matchs du jour ESPN API

Récupération automatique du calendrier ATP/WTA du jour — tournois, surfaces, rounds, têtes de série.

02
Profils joueurs mis à jour Live

ELO dynamique, classement ATP/WTA, forme récente (5/10/20 matchs), fatigue, statistiques de service recalculées automatiquement.

03
Prédiction XGBoost ML

Le modèle génère une probabilité calibrée de victoire. Platt scaler appliqué pour corriger le biais systématique.

04
Comparaison cotes Pinnacle no-vig Odds API

Les cotes brutes sont converties en probabilités implicites sans marge bookmaker. L'edge est la différence entre les deux.

05
Signals de décision

Chaque match reçoit un signal selon l'amplitude de l'edge détecté et la qualité des données disponibles.
